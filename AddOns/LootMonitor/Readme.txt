使用说明：
1. 命令/lm 或/lootmonitor 开启/关闭设置窗口
2. 检查所有按钮检查当前团队插件安装情况，灰色为没有插件，橙红色为团队通报者
3. 自动反和谐（包括语言和坟包）
4. 支持Party和Raid
5. Raid模式可选输出至Raid_Warning频道
6. 仅自身安装插件时，可以关闭通报功能
7. 下拉菜单可选择通报物品等级
8. 兼容1.38+版本
9. Hubbotu提供了俄文翻译。
10. 添加小地图按钮。
11. ESC关闭窗口。
12. 保存窗口位置。
13. 支持LDB。

Readme:
1. Slash Command /lm or /lootmonitor to open/close LootMonitor Window.
2. Check All Button for check party/raid group member who installed this addon.
3. Show State Button to print members who installed/not installed this addon, and who will announce loots.
4. Support Party and Raid.
5. In Raid, you can announce loots to Raid_Warning(RW) channel.
6. Only you installed this addon, you can check Single Disable Checkbox to disable announce.
7. You can use Dropmenu select which quality item(s) you will report to announcer.
8. Can work with Version 1.38+ LootMonitor Addon.
9. Thanks to Hubbotu for translate this addon to Russian.
10. Add Minimap Button.
11. Press Escape to close Set/Output Window.
12. Save Set/Output Window position.
13. Support LDB.
